# Idmar Ramos Junior personal website

Just another personal website. Feel free to fork, use and/or modify.

## Stack

- [NodeJS](https://nodejs.org/)
- [pnpm](https://pnpm.io/)
- [Vite](https://vitejs.dev/)
- [Vue 3](https://vuejs.org/)
- [Typescript](https://www.typescriptlang.org/)
- [Vitest](https://vitest.dev/)

## Setup

1. `pnpm install`
1. `pnpm dev`

## Build & Deploy

1. `pnpm build`
1. `pnpm preview`
1. Use the newly created `./dist/` folder to deploy

## Tests

Tests are written with [Vitest](https://vitest.dev/) and [@vue/test-utils](https://test-utils.vuejs.org/).

Run once:

```
pnpm test
```

Run in watch mode during development:

```
pnpm test:watch
```

## Google Analytics

1. Duplicate [googleAnalyticsScript-example.js](googleAnalyticsScript-example.js) and rename it to `googleAnalyticsScript.js`
1. Update `YOUR-GA-ID` with your Google Analytics ID
